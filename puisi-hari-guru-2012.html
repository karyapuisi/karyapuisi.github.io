1. Jika hari ini senin , maka 2012 hari lagi adalah


2012 = (7 x 287) + 3
= 2009 + 3
Jadi 2012 hari lagi adalah hari KAMIS2012/7=287 sisa 3 hari
287×7=2009

2009 hari lagi setelah hari senin=hari senin

2012 hari setelah hari senin=hari senin+3 hari
                                                =hari kamis

2. Jika hari ini hari senin.Maka 2012 hari yang akan datang adalah hari...


Jika hari ini hari Senin maka 2012 hari yang akan datang adalah hari Kamis.

Pembahasan

Diketahui:

Hari ini = hari Senin

Hari yang akan datang = 2012 hari

Ditanya:

hari ke-2012 = .... ?

Jawab:

Untuk mencari hari ke-n maka harus dipahami dulu bahwa dalam 1 minggu terdapat 7 hari, yang terdiri dari hari Senin, Selasa, Rabu, Kamis, Jumat, Sabtu, Minggu. Maka, untuk mencari hari ke-n di mana n tersebut diketahui dicari dengan membagi sejumlah hari ke-n dengan 7 (jumlah hari). Jika hasil baginya = 0 maka hari ke-n adalah hari yang sama dengan hari ini. Jika hasil baginya > 1 maka sisa dari hasil bagi dijumlahkan ke hari ini dan menghitung maju harinya.

Untuk lebih jelasnya, yuk disimak penyelesaian soal di atas berikut:

2012 : 7 = 2009 ⇒ sisa = 3

Maka sisa 3 digunakan untuk menghitung maju hari ini. Karena hari ini Senin, maka hari ke-1 setelah Senin adalah Selasa, hari ke-2 setelah Senin adalah Rabu, dan hari ke-3 setelah hari Senin adalah Kamis.

Jadi, 2012 hari yang akan datang adalah hari Kamis.

26. jika hari ini hari kamis maka 2 pangkat 2012 lagi hari apa?


Jika hari ini hari kamis maka 2 pangkat 2012 lagi adalah hari senin. Caranya adalah dengan cari pola untuk 2ⁿ.  

1 minggu = 7 hari

Pembahasan  

Misal kita akan mencari pola untuk 2ⁿ untuk n = 1, 2, 3, 4, ….

2¹ = 2 ⇒ 2 hari setelah kamis adalah sabtu 2² = 4 ⇒ 4 hari setelah kamis adalah senin 2³ = 8 ⇒ 8 hari = 1 minggu 1 hari = 1 hari setelah kamis adalah jumat 2⁴ = 16 ⇒ 16 hari = 2 minggu 2 hari = 2 hari setelah kamis adalah sabtu 2⁵ = 32 ⇒ 32 hari = 4 minggu 4 hari = 4 hari setelah kamis adalah senin 2⁶ = 64 ⇒ 64 hari = 9 minggu 1 hari = 1 hari setelah kamis adalah jumat 2⁷ = 128 ⇒ 128 hari = 18 minggu 2 hari = 2 hari setelah kamis adalah sabtu

dan seterusnya

Jadi berdasarkan pola diatas, harinya selalu berulang tiap tiga hari yaitu sabtu, senin, jumat

Jika hari ini hari senin, maka 2ⁿ hari lagi adalah

Hari sabtu jika n dibagi 3 bersisa 1 Hari senin jika n  dibagi 3 bersisa 2 Hari jumat jika n habis dibagi 3 (bersisa 0)

Karena 2012 dibagi 3 adalah bersisa 4 yaitu

2012 = 670 × 3 + 2  

artinya 2012 dibagi 3 sama dengan 670 bersisa 2,  

sehingga jika hari ini adalah hari kamis maka 2²⁰¹² hari lagi adalah hari senin
