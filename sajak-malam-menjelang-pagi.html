26. Quiiz lumayan gampang Suhu di suatu daerah -8°C pada siang hari, pada malam hari suhu turun 3°C, menjelang pagi, suhu turun lagi 3°Cpertanyaan: Hitunglah selisih suhu malam hari dengan suhu menjelang pagi!(memakai cara)#GABOLE PAKE KALKULATOR​


Jawaban:

Diketahui : Suhu awal -8°C ( Siang Hari )

Suhu ( Malam Hari ) Turun 3°C

Suhu ( Pagi Hari ) Turun 3°C

Ditanya : Selisih suhu malam hari Menjelang pagi?

Dijawab :

A. Dicari dulu suhu sebenarnya pada pagi hari :

= ( Suhu Siang - Suhu malam ) - Suhu pagi

= ( - 8°C - 3°C = - 11°C )

= - 11°C - 3°C

= - 14°C ( Suhu Pada pagi hari )

B. Selisih suhu malam dengan suhu menjelang pagi hari :

Suhu malam - Suhu pagi

= - 11°C - ( - 14°C )

= - 11°C + 14°C

= 3°C

Maka selisih Suhu malam hari dengan suhu menjelang pagi hari adalah 3°C
