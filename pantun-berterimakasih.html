DETAIL JAWABAN

Kelas : X SMA

Mapel : Matematika Wajib

Bab : Nilai Mutlak

Indikator : Persamaan Nilai Mutlak

1.

| x - 2 | = 3

(x - 2) = –3 atau (x - 2) = 3

x = –3 + 2 atau x = 3 + 2

x = –1 atau x = 5

HP = { –1 , 5 }

2.

| 2x - 3 | = 5

(2x - 3) = –5 atau (2x - 3) = 5

2x = –5 + 3 atau 2x = 5 + 3

2x = –2 atau 2x = 8

x = –1 atau x = 4

HP = { –1 , 4 }

3.

| 2x - 1 | < 7

–7 < 2x - 1 < 7

–7 + 1 < 2x - 1 + 1 < 7 + 1

–6 < 2x < 8

–6 ÷ 2 < 2x ÷ 2 < 8 ÷ 2

–3 < x < 4

HP = { –3 < x < 4 }

4.

| 4x + 2 | ≥ 6

(4x + 2) ≤ –6 atau (4x + 2) ≥ 6

4x ≤ –6 - 2 atau 4x ≥ 6 - 2

4x ≤ –8 atau 4x ≥ 4

x ≤ –2 atau x ≥ 1

HP = { x ≤ –2 atau x ≥ 1 }

5.

| 2x + 1 | > | x - 7 |

« kuadratkan kedua ruas »

(2x + 1)² > (x - 7)²

(2x + 1)² - (x - 7)² > 0

« INGAT : p² - q² = (p - q)(p + q) »

( (2x + 1) - (x - 7) )( (2x + 1) + (x - 7) ) > 0

( 2x + 1 - x + 7 )( 2x + 1 + x - 7 ) > 0

( x + 8 )( 3x - 6 ) > 0

Nilai x pembuat nol :

› x + 8 = 0 => x = –8

› 3x - 6 = 0 => x = 2

Pada garis bilangan, untuk interval :

» x < –8, ( x + 8 )( 3x - 6 ) bernilai [ > 0 ]

» –8 < x < 2, ( x + 8 )( 3x - 6 ) bernilai [ < 0 ]

» x > 2, ( x + 8 )( 3x - 6 ) bernilai [ > 0 ]

HP = { x < –8 ∪ x > 2 }

18. Mengapa kita harus berterimakasih kepada guru


Jawaban:

Kita harus berterimakasih kepada guru karena peran mereka yang penting dalam pendidikan dan perkembangan kita sebagai individu. Berikut adalah beberapa alasan mengapa kita harus menghargai dan berterima kasih kepada guru.

Contoh:

Pengetahuan dan keterampilan: Guru adalah mereka yang memberikan pengetahuan dan keterampilan kepada kita. Mereka membantu membentuk pikiran dan pemahaman kita tentang dunia di sekitar kita. Guru membantu kita menguasai subjek-subjek yang beragam dari matematika hingga bahasa, dan memberi kita landasan yang kuat untuk masa depan kita.Pembimbingan dan inspirasi: Guru tidak hanya memberikan pengetahuan secara kelasikal, tetapi juga memberikan bimbingan dan inspirasi kepada murid-muridnya. Mereka memberikan motivasi, dorongan, dan pedoman dalam pengembangan akademik, intelektual, dan emosional kita. Guru sering kali menjadi teladan atau panutan dan menginspirasi kita untuk mencapai potensi maksimal kami.Pengembangan karakter: Guru berperan dalam membantu membentuk karakter dan nilai-nilai kita. Mereka mengajarkan etika, integritas, tanggung jawab, kerjasama, dan berbagai keterampilan sosial lainnya. Guru juga membantu membentuk sikap dan persepsi kita tentang dunia, membantu kita menjadi warga yang bertanggung jawab dalam masyarakat.Mendorong pertumbuhan pribadi: Guru berperan dalam mendorong pertumbuhan pribadi kita. Mereka mendukung kita dalam menghadapi tantangan, mengatasi kelemahan, membangun kepercayaan diri, dan mengembangkan bakat dan minat kita. Guru mendengarkan, memberi umpan balik, dan memberi arahan yang konstruktif untuk membantu kita tumbuh dan berkembang sebagai individu.Dedikasi dan pengorbanan: Sebagian besar guru memiliki dedikasi dan pengorbanan yang luar biasa terhadap pekerjaan mereka. Mereka bekerja keras untuk memberikan pendidikan terbaik bagi murid-muridnya, bahkan dengan waktu dan energi yang melebihi kewajiban mereka. Menghargai dan berterima kasih kepada guru adalah cara kita mengakui upaya dan pengorbanan mereka dalam menyebarkan pengetahuan dan membantu kami tumbuh.

Kita perlu selalu mengingat bahwa pendidikan adalah batu loncatan untuk masa depan yang sukses, dan itu tidak akan mungkin tanpa guru yang hebat. Mengucapkan terima kasih kepada guru adalah penghormatan kepada mereka dan pengakuan akan peran penting yang mereka mainkan dalam hidup kita.
