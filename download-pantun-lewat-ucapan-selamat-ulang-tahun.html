37. cara mendownload aplikasi lewat pc/laptop


mapel : ti

kelas : 10

materi : sistem operasi dan perangkat lunak

kata kunci : mendownload aplikasi

kode soal : 11

==========================================

[tex] software [/tex]

untuk mendownload aplikasinya :

1. pergi ke situs web aplikasinya. perlu kamu ketahui kalau kamu mengunduh file yang berisi perangkat perusak (virus, spyware, dll) dapat merusak perangkat sistem operasimu

2. biasanya ada tombol "download". klik download.

3. untuk sistem operasi windows, file yang digunakan untuk melakukan instalasi program adalah file berekstensi .exe ataupun .msi. tergantung kepada aplikasinya. biasanya ada label setup.

4. ketika ada jendela kecil yang menampilkan file yang diunduh, klik ok.

5. ketika sudah selesai mengunduh file instalasinya, klik dua kali pada filenya hingga muncul jendela baru bertulliskan "selamat datang" / "welcome"

6. untuk melanjutkan instalasi programnya, ikuti instruksi yang diberikan

39. ada yang tau cara melewati survey saat download


Survey yang gak bisa diclose, scroll down, atau dilewatin itu adalah javascript. Supaya bisa dilewatin, ya nonaktifkan javascriptnya.

Kalau di Firefox, bisa dari Edit - Preferences - Content - unchecklist bagian 'Enable Javascript' - refresh page yang ada surveynya.

Kalau di Chrome, Costumize and control Google Chrome (ikon kunci, biasanya di sisi kanan address bar) - Preferences - Content settings - di bagian Javascript pilih 'Don't allow any site to run JavaScript' - refresh page yang ada surveynya.

Semoga membantu, di browser komputer aku sih bisa u-u kalau gak salah ada aplikasinya, namanya File Ice Downloader- di internet ada linknya.

(*) setelah selesai download, mendingan diaktifin lagi javascriptnya, supaya konten web lain bisa ditampilkan dengan baik :)
