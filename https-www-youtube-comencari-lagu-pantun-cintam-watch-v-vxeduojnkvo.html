5. apakah perbedaan www dan https?​


HTTPS atau hyper-text transfer protocol secure adalah serangkaian peraturan dalam komunikasi antara 2 system seperti komputer client dan web server. HTTPS menambahkan protocol secure untuk mengamankan transfer hyper-text 2 system tersebut dengan cara enkripsi dan dekripsi. Sedangkan WWW atau World Wide Web adalah prefix dari domain name system atau DNS untuk menyatakan bahwa website yang diakses menggunakan protocol HTTP/HTTPS dan dapat diakses melalui internet. WWW dapat digunakan sebagai penanda bahwa URL yang diakses adalah HTTP/HTTPS. Hal ini dapat membantu untuk membedakan file server yang menggunakan FTP dengan web server yang menggunakan HTTP/HTTPS sebagai transfer protocolnya.

Menghapus konten yang diklaim dari video

Jika terdapat klaim hak cipta di video Anda, kemungkinan akan ada pembatasan terkait tempat penayangan video atau kemampuan untuk memonetisasi video tersebut.

Jika klaim dikenakan untuk audio dalam video, Anda memiliki beberapa opsi untuk menghapus batasan ini:

Memangkas bagian video yang dikenai klaim hak cipta

Mengganti lagu dengan audio dari Koleksi Audio kami

Membisukan suara lagu

Jika klaim dikenakan untuk hal lain, Anda dapat:

Memangkas bagian video yang dikenai klaim hak cipta

Catatan: Jika durasi video Anda lebih dari 6 jam, perubahan mungkin tidak dapat disimpan. Selain itu, jika channel Anda tidak tergabung dalam Program Partner YouTube, Anda mungkin tidak dapat menyimpan perubahan jika video Anda memiliki lebih dari 100.000 penayangan.

Menemukan video yang dikenai klaim hak cipta

Login ke YouTube Studio.

Dari menu kiri, klik Konten.

Filter untuk Klaim hak cipta.

Di kolom “Batasan”, arahkan kursor ke “Klaim hak cipta”, lalu klik LIHAT DETAIL.

Klik PILIH TINDAKAN dan pilih tindakan yang akan dilakukan:Catatan:

Jika Anda melakukan pengeditan, waktu pemrosesan bisa berbeda-beda.

Anda tidak dapat melakukan pengeditan lain saat video sedang diproses, tetapi Anda dapat menutup jendela. Status video akan tetap seperti saat ini sampai pemrosesan selesai.

Ketika pemrosesan selesai, muat ulang halaman jika pengeditan berhasil dan klaim telah dilepas.

Mengurungkan perubahan lagu

Untuk mengurungkan penggantian lagu atau membunyikan suara lagu:

Login ke YouTube Studio.

Dari menu kiri, klik Konten.

Klik thumbnail video yang ingin diedit.

Di menu sebelah kiri, klik Editor.

Pilih More lalu Kembalikan ke versi asli untuk menghapus semua perubahan yang sudah Anda buat pada video.

Besaran merupakan suatu pengukuran dalam fisika dan satuan adalah unit yang terukurnya .

I.Besaran : Sesuatu yang dapat diukur, memiliki nilai dan satuan  

Pembagian Besaran  

1.Berdasar Satuan  

a.Besaran Pokok : Besaran yang satuan sudah terdefinisikan dan berdiri sendiri  

Contoh :  

( J I Wa Ku S M P )  

Jumlah zat ( mol ), Intensitas cahaya ( Cd ), Waktu ( s ), Kuat arus ( A ), Suhu ( K ), Massa ( Kg ) dan Panjang ( m )  

b.Besaran Turunan : Besaran yang satuannya diturunkan dari satuan besaran pokok  

Contoh :  

Luas ( m2 ), Volume ( m3 ), Massa jenis ( Kg/m3 ), Kecepatan ( m/s ), Percepatan ( m/s2 ), Gaya ( N = kgm/s2 ), Energi ( J = Nm = Kgm2/s2 ), Daya ( Watt = J/s = Kgm2/s3 ), Tekanan ( N/m2 = Kg/ms2 ), Momentum ( Kgm/s ), Impuls ( Kgm/s ), Usaha (J)  

2.Berdasar Arah  

a.Besaran Skalar : Besaran yang hanya memiliki nilai saja  

Contoh : Semua besaran pokok termasuk besaran skalar, Luas, Volume, Kelajuan, Perlajuan, Massa Jenis, Energi, Daya dsb  

b.Besaran Vektor : Besaran yang memiliki nilai dan arah  

Contoh : Perpindahan, Kecepatan, Percepatan, Gaya, Momentum dsb  

II.Dimensi  

Merupakan gambaran bagaimana besaran tersebut tersusun atas besaran-besaran pokok.  

Fungsi dimensi : untuk menguji benar / setara tidaknya suatu persamaan ( Persamaan yang benar secara dimensi belum tentu benar secara fisis )  

Dimensi dari besaran – besaran pokok :  

Jumlah zat [ N ], Intensitas cahaya [ J ], Waktu [ T ], Kuat arus [ I ], Suhu [ θ ], Massa [ M ] dan Panjang [ L ]

Pembahasan  

Pada soal tidak ada gambar, namun pada intinya perhatikan gambar.

-jika jarak antara titik oli konstan maka benda bergerak dengan kecepatan konstan

-jika jaraknya semakin jauh, maka maka benda bergerak dipercepat

-jika jaraknya semakin dekat, maka maka benda bergerak diperlambat
