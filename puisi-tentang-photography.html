30. Apa perbedaan antara sensor photography dengan sensor elektronik dan berikan contonya sensor tetsebut??


Jawaban:

perbedaan sensor fotografik dan sensor elektromagnetik adalah

Sensor fotografik

• proses perekamannya berlangsung secara kimiawi.

• pemotretan dilakukan dari pesawat udara atau wahana, fotonya disebut foto udara

• pemotretan dilakukan dari antariksa atau menggunakan satelit,

• hasil fotonya disebut citra satelit atau foto satelit.

Sensor elektronik

• Sensor menggunakan tenaga elektrik berbentuk sinyal elektrik.

• Alat penerima dan perekamannya berupa pita magnetik

• Pemrosesan agar menjadi citra

• Sinyal elektrik direkam pada pita magnetik dan diproses menjadi data visual maupun data digital.

Penjelasan:

Sensor merupakan bagian untuk merekam data. Disini terdapat dua jenis sensor yaitu sensor elektronik dan sensor fotografik. Yaitu;

- Sensor fotografik yaitu proses perekamannya berlangsung secara kimiawi, pemotretan dilakukan dari antariksa atau menggunakan satelit fotonya disebut citra satelit.

- Sensor elektronik yaitu sensor yang menggunakan tenaga elektrik dalam bentuk sinyal elektrik, alat penerimanya berupa pita magnetik atau detector.

- sensor elektromagnetik adalah sensor bertenaga elektrik yang beroperasi pada spektrum yang lebih luas.
