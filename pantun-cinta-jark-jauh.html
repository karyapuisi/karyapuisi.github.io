1. antar muka yang di gunakan pada aplikasi telnet/ssh aksess jarK JAUH ADALAH


Antar muka yang digunakan pada aplikasi telnet atau ssh jarak jauh adalah menggunakan antar muka teks. Telnet merupakan kepanjangan dari Tellecomunication Network. Telnet merupakan salah satu komunikasi antar komputer. Antar muka teks adalah antar muka yang hanya berisikan teks dan biasanya hanya menggunakan keyboard sebagai inputannya. Tampilannya mirip dengan command prompt pada windows atau terminal pada lInux.

Seiring dengan perkembangan teknologi, sekarang apabila kita ingin mengakses komputer lain menggunakan telnet atau ssh kita dapat menggunakan aplikasi yang bernama putty. Aplikasi ini merupakan aplikasi yang cukup populer. Namun sebenarnya akses menggunakan telnet masih dapat menggunakan command prompt pada windows atau untuk pengguna linux dapat menggunakan terminal untuk mengakses aplikasi telnet.

Dengan menggunakan telnet seseorang dapat mengakses komputer lain tanpa harus berhadapan fisik dengan komputer yang diakses, atau bisa disebut dengan remote. Namun apabila menggunakan telnet harap berhati-hati karena keamanan masih kurang. Solusi dari permasalahan tersebut adalah menggunakan SSH. Dengan SSH keamanan akses remote komputer  menjadi lebih terjamin keamanannya.

Demikian pembahasan mengenai antar muka pada aplikasi telnet/ssh yang menggunakan antar muka teks. Untuk membaca materi lainnya mengenai remote komputer dapat dibaca pada link berikut

20. Jika sebuah bola memiliki luas permukaan 2.464cm² berapakah jari jark bola tersebut?


jari jari bola

LP = 4πr²

2.464 = 4 × 22/7 × r²

r = √2.464 ÷ 4 × 22/7

r = √2.464 ÷ 88/7

r = √2.464 × 7/88

r = √17.248/88

r = √196

r = 14 cm

Diketahui:

bola

L = 2.464 cm²

Ditanya: r

Penyelesaian:

L = 4πr²

2.464 = 4 x 3,14 x r²

2.464/4 = 3,14r²

3,14r² = 616

r² = 616/3,14

r² = 196,178

r² ≈ 196

r ≈ √196

r ≈ 14 cm

Jadi, jari-jari bola tersebut adalah 14 cm.

disini diketahui:

Jarak pada peta = 8 CmJarak sebenarnya = 320 KM

Penyelesaian:

Skala = Jarak peta : jarak sebenarnya

         = 8 cm : 320 KM

(Semua diubah menjadi cm karena jarak pada peta cm)

         = 8 cm : 32.000.000 cm

(disederhanakan)

         = 1 : 4.000.000 cm

Jadi, Skalanya adalah 1 : 4.000.000

Mohon maaf jika ada kesalahan
